## Hosting
1. In `.env` file set:
    * `DOMAIN` - Website address (should start with `http://` or `https://`)
    * `PORT` - Website port
    * `WEBHOOK` - Your Discord webhook link
2. Run `npm i`
3. Run `npm run start`
4. Visit your browser for API
